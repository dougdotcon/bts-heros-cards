# BTS Hero Cards

![Banner](./img/background.jpg)

**BTS Hero Cards** is an interactive website dedicated to transforming the members of BTS into true heroes, each with their unique abilities. The project celebrates the dedication, talent, and team spirit that make BTS a special group. Browse through the cards, discover more about each member, and witness their heroic transformations.

- **3D Card Animations:** Hover or click on member cards to see their heroic transformations with 3D flip animations.
- **Info Modals:** Interacting with a card displays a modal with detailed information about the selected hero.
- **Informative Sections:** Includes sections about the project, an image gallery, and a contact form for feedback.
- **Responsive Design:** The layout adapts to various screen sizes, ensuring a great experience on mobile and desktop devices.
- **Accessibility:** Keyboard navigation and screen reader friendly, with visible focus indicators and proper ARIA attributes.

## ğŸ“ Usage Guide

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).
- A local server to view HTML files (optional, but recommended).

### Installation

1. **Clone the Repository:**

   bash
   git clone https://github.com/seu-usuario/bts-heros-cards.git
   

2. **Navigate to the Project Directory:**

   bash
   cd bts-heros-cards
   

3. **Open `index.html` in your Browser:**

   - You can open the file directly or use a local server for the best experience.

   **Using a Local Server with VS Code:**

   - Install the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension.
   - Open the project in VS Code.
   - Right-click on `index.html` and select "Open with Live Server".

### How to Use

- **Navigation:**
  - Use the navigation bar to access the **Home**, **About**, and **Gallery** sections.
  - Click on a member card to view their detailed hero profile in a modal.
- **Contact:**
  - Navigate to the Contact page to submit feedback via the form.
